# NeuralSnake
A machine learning model attempts to simulate the game of snake! If fed enough data, can a model take user inputs and infer what the next game state should be?

====================================================================

**To run this project:**

1) run generate_truth_dataset.py
2) run the training script for your preferred model (baseline, transformer, unet)
3) set MODEL in play_neural_snake.py to the name of your preferred model and run play_neural_snake.py

To get performance metrics, run your preferred evaluation script or generalisability experiments. 
Note: this will require model weights for all models (included in this REPO)
